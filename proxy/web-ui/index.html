Midjourney Studio

这里把 Midjourney 的核心工作流集中在一起：首图生成、U1-U4 放大、V1-V4 变化、Reroll 重抽、以及编辑（加元素 / 扩图 / 改比例）。全部默认走 Relax 模式，价格按 ¥1.00 展示。

怎么用

  • 出图：在下方输入提示词、比例，点击“生成 Midjourney 图像”。
  • 放大 / 变化：首图完成后，下方会自动出现 U1-U4 / V1-V4 / REROLL 按钮。
  • 编辑：可把当前结果图一键带入编辑面板，继续加元素、改比例、扩范围。
  • 作品库联动：作品库里的图片也可以一键带入 MJ 编辑，无需手动复制 URL。

控制说明

  • U1-U4：放大四宫格中对应位置的图。
  • V1-V4：基于对应位置继续出变化版本。
  • REROLL：保留同一提示词重新抽一组。
  • MJ 编辑：适合做加元素、扩图、改构图、改比例。
