Midjourney Studio

Core Midjourney workflows in one place: image generation, U1-U4 upscale, V1-V4 variations, Reroll, and editing (add elements / outpaint / change aspect ratio). Defaults to Relax mode, priced at ¥1.00.

How to Use

  • Generate: Enter a prompt and aspect ratio below, then click “Generate Midjourney Image”.
  • Upscale / Variate: After the initial image, U1-U4 / V1-V4 / REROLL buttons appear automatically.
  • Edit: Send the current result to the edit panel to add elements, change aspect, or expand the frame.
  • Gallery Integration: Images from the gallery can be sent directly to MJ Edit — no need to copy URLs.

Controls

  • U1-U4: Upscale the corresponding quadrant image.
  • V1-V4: Generate variations based on the corresponding quadrant.
  • REROLL: Re-roll with the same prompt for a new set.
  • MJ Edit: Add elements, outpaint, recompose, or change aspect ratio.
