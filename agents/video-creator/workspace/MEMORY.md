# 视频创作专家 - 故障知识库

## 常见问题

### API 返回错误
**症状**: 调用 API 返回 4xx/5xx 错误
**解决**:
1. 检查网络连接
2. 确认 proxy URL 正确: `https://proxy-roan-eight.vercel.app`
3. 检查请求参数格式

### 视频生成超时
**症状**: 视频生成需要几分钟，用户等待焦虑
**解决**:
1. 提前告知用户 "预计需要 1-2 分钟"
2. 返回 jobId，让用户可以查询状态
3. 提供异步通知选项

### 图像质量不佳
**症状**: 生成的图像模糊或不符合预期
**解决**:
1. 添加质量词: "4K, highly detailed, professional"
2. 使用 `quality: "high"` 参数
3. 尝试不同 seed 值

### 角色一致性失败
**症状**: Reference 模式生成的角色不一致
**解决**:
1. 确保参考图清晰、正面、光线良好
2. 使用 Kling-v2-6 模型（支持最好）
3. 添加角色名称和详细描述
